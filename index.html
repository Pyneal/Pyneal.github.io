一份简短的博客搭建记录

1.安装Node.js

官网下载安装

命令行执行 npm -v 确认是否安装成功

换个源加速后续下载和安装，执行 npm install -g cnpm --registry=https://registry.npmmirror.com

2.安装Hexo

cnpm安装Hexo框架，执行 cnpm install -g hexo-cli

hexo -v 确认是否安装成功

3.本地写博客

安装git ，比cmd方便，后续推送云端也需要

新建文件夹，打开 git bash ，输入hexo init ，运行完毕后文件夹内会出现下面这些文件（windows查看文件是dir，linux查看文件是ls）

创建新博客

生成静态网页

运行服务器，本地显示

部署到远端

注意云端仓库代码和本地并不一致，因此不需要 git push 工作流
