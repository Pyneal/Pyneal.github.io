安装Node.js

官网下载安装

命令行执行 npm -v 确认是否安装成功

换个源加速后续下载和安装，执行 npm install -g cnpm --registry=https://registry.npmmirror.com

安装Hexo

cnpm 安装Hexo框架，执行 cnpm install -g hexo-cli

hexo -v 确认是否安装成功

新建文件夹，打开 git bash ，输入hexo init ，运行完毕后文件夹内会出现下面这些文件（windows查看文件是dir，linux查看文件是ls）

生成静态网页

运行服务器，本地显示

部署到远端

如果生成静态网页遇到问题，可以尝试清空 Hexo 运行产生的临时 / 成品静态页面文件，再重新生成部署

注意云端仓库代码和本地并不一致，因此不需要 git push 工作流

优化显示

默认采用的主题是 landscape ，可以换成 butterfly

在根目录下执行 npm install hexo-theme-butterfly ，若升级则运行 npm update hexo-theme-butterfly

修改根目录下的 _config.yml：theme: butterfly

使用主题还需要额外的渲染器：npm install hexo-renderer-pug hexo-renderer-stylus --save

根目录下新建 _config.butterfly.yml ，将安装主题中的 _config.yml 内容复制进入

编辑 _config.butterfly.yml 时需注意，Hexo 的根目录对于静态资源来说是 source 文件夹，而非整体文件夹。因此在 .yml 中写 /img/me.png 实际上要放到 /source/img/me.png 才能被正确渲染
